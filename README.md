# liblogger

The c-logger logging library packaged up with a Visual Studio solution for
building on Windows for local use (as a git submodule).

This repository is __not__ affiliated with the official c-logger maintainers.

The current version of the c-logger code may be found at it's repository:

* [c-logger](https://github.com/yksz/c-logger)

Please support the official release.
